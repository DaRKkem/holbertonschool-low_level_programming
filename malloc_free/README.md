# malloc and Free

Dynamic memory allocation — creating arrays, duplicating strings, concatenating, and building 2D grids on the heap.

---

## Tasks

- **0-create_array.c**: Allocate array of chars initialized with a specific char
- **1-strdup.c**: Duplicate a string (like strdup)
- **2-str_concat.c**: Concatenate two strings
- **3-alloc_grid.c**: Allocate 2D grid of integers initialized to 0
- **4-free_grid.c**: Free a 2D grid

## Key Concepts

malloc(), free(), dynamic memory, NULL-checking after allocation, 2D dynamic arrays, string duplication
