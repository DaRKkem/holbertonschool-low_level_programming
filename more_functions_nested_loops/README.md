# More Functions and Nested Loops

Advanced character checks, geometry drawing, and classic algorithm problems.

---

## Tasks

- **0-isupper.c**: Check uppercase character
- **1-isdigit.c**: Check digit character
- **2-mul.c**: Multiply two integers
- **3-print_numbers.c**: Print 0-9
- **4-print_most_numbers.c**: Print 0-9 except 2 and 4
- **5-more_numbers.c**: Print 0-14 ten times
- **6-print_line.c**: Draw straight line
- **7-print_diagonal.c**: Draw diagonal line
- **8-print_square.c**: Draw square
- **9-fizz_buzz.c**: FizzBuzz implementation
- **10-print_triangle.c**: Draw triangle

## Key Concepts

isupper, isdigit, nested loops for geometric patterns, FizzBuzz algorithm, _putchar-based shape drawing
