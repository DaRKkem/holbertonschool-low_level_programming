# Variadic Functions
![C](https://img.shields.io/badge/C-99-blue) ![Betty](https://img.shields.io/badge/style-Betty-green) ![License](https://img.shields.io/badge/License-Holberton-yellow)

Variable-argument functions using the stdarg.h library — flexible parameter handling and type-generic printing.

---

## Tasks

- **0-sum_them_all.c**: Sum variable number of integers
- **1-print_numbers.c**: Print numbers with separator
- **2-print_strings.c**: Print strings with separator
- **3-print_all.c**: Generic printer for c/i/f/s format specifiers

> **Example:** `int sum_them_all(unsigned int n, ...) { va_list ap; va_start(ap, n); ... }`

## Key Concepts

va_list, va_start, va_arg, va_end, variadic function prototypes (...), type-generic formatting
