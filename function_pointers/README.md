# Function Pointers
![C](https://img.shields.io/badge/C-99-blue) ![Betty](https://img.shields.io/badge/style-Betty-green) ![License](https://img.shields.io/badge/License-Holberton-yellow)

Passing functions as arguments, callbacks, and building a calculator with operation dispatch via function pointers.

---

## Tasks

- **0-print_name.c**: Call a function with a name string
- **1-array_iterator.c**: Apply function to each array element
- **2-int_index.c**: Search array with comparison function
- **3-calc.h**: Calculator header
- **3-op_functions.c**: Calculator operation functions (+, -, *, /, %)
- **3-get_op_func.c**: Get function for operator
- **3-main.c**: Calculator main

> **Example:** `void array_iterator(int *array, size_t size, void (*action)(int)) { ... }`

## Key Concepts

Function pointer syntax, callbacks, array iteration, search with comparator, operator dispatch table
