Introduction to C — compilation pipeline, basic I/O functions, and data type sizes.

---

## Tasks

- **0-preprocessor**: Script that runs the C preprocessor
- **1-compiler**: Script that compiles without linking (assembly output)
- **2-assembler**: Script that generates assembly code
- **3-name**: Script that compiles with custom output name
- **4-puts.c**: Using puts() to print a string
- **5-printf.c**: Using printf() to print formatted output
- **6-size.c**: Using sizeof() to print sizes of char, int, long, long long, float

> **Example:** `gcc -Wall -Werror -Wextra -pedantic 4-puts.c -o hello && ./hello`

## Key Concepts

Preprocessing, compilation, assembly, linking, puts(), printf(), sizeof operator
