# Variables, if, else, while

Control flow fundamentals — conditional statements, loops, and basic variable usage in C.

---

## Tasks

- **0-positive_or_negative.c**: Random number — print positive/zero/negative
- **1-last_digit.c**: Print last digit of a random number
- **2-print_alphabet.c**: Print alphabet in lowercase
- **3-print_alphabets.c**: Print alphabet in lowercase then uppercase
- **4-print_alphabt.c**: Print alphabet except q and e
- **5-print_numbers.c**: Print base-10 digits
- **6-print_numberz.c**: Print digits using putchar
- **7-print_tebahpla.c**: Print alphabet in reverse
- **8-print_base16.c**: Print hexadecimal digits
- **9-print_comb.c**: Print combinations of digits

> **Example:** `int n; srand(time(0)); n = rand(); if (n > 0) printf("positive\n");`

## Key Concepts

if/else if/else, while loops, for loops, putchar, rand()/srand(), ASCII manipulation
