# C Programming
![C](https://img.shields.io/badge/C-99-blue) ![Betty](https://img.shields.io/badge/style-Betty-green) ![License](https://img.shields.io/badge/License-Holberton-yellow)

This repository contains the C Programming curriculum at Holberton School. It progresses from basic syntax and compilation through memory management, data structures, recursion, file I/O, and advanced function pointer techniques. Each module builds upon the previous, forming a comprehensive foundation in low-level programming.

## Sub-Projects

| Module | Description |
|--------|-------------|
| hello_world | Compilation pipeline, puts/printf, sizeof |
| variables_if_else_while | Conditionals, loops, putchar |
| functions_nested_loops | Function declarations, nested loops, character classification |
| more_functions_nested_loops | Advanced functions, isupper/isdigit, FizzBuzz, geometric shapes |
| pointers_arrays_strings | Pointers, arrays, string manipulation, memory functions (memset/memcpy), 2D arrays |
| recursion | Recursive functions (string traversal, factorial, power, sqrt, prime check) |
| malloc_free | Dynamic memory allocation, malloc/free, 2D grids |
| more_malloc_free | Advanced malloc (exit(98)), custom calloc, string_nconcat, array_range |
| function_pointers | Function pointers, callbacks, array_iterator, calculator dispatch |
| variadic_functions | va_list, va_start/va_arg/va_end, type-generic printing |
| singly_linked_lists | Linked list creation, traversal, insertion, freeing |
| doubly_linked_lists | (Team project) Doubly linked lists CRUD |
| structures_typedef | Struct definition, typedef, deep copy, memory management |
| file_io | POSIX file I/O (open/read/write/close), file descriptors, cp clone |
| argc_argv | Command-line arguments (argc/argv), atoi, input validation |
