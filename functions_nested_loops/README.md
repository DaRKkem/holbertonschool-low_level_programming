# Functions and Nested Loops

Custom function declarations, nested loop patterns, character classification, and mathematical helpers.

---

## Tasks

- **_putchar.c**: Custom putchar implementation
- **0-putchar.c**: Print _putchar
- **1-alphabet.c**: Print alphabet 10 times
- **2-print_alphabet_x10.c**: Alphabet x10 with nested loops
- **3-islower.c**: Check lowercase character
- **4-isalpha.c**: Check alphabetic character
- **5-sign.c**: Print sign of a number
- **6-abs.c**: Compute absolute value
- **7-print_last_digit.c**: Print last digit
- **8-24_hours.c**: Print Jack Bauer's 24-hour clock
- **9-times_table.c**: Print 9 times table
- **10-add.c**: Add two integers
- **11-print_to_98.c**: Print numbers to 98

## Key Concepts

Function prototypes, nested for loops, character classification (islower/isalpha), absolute value, modular arithmetic, formatted looping
