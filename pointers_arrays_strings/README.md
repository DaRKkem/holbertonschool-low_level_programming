# Pointers, Arrays, and Strings

Comprehensive coverage of pointer arithmetic, string manipulation, memory operations, and 2D arrays.

---

## Tasks

**Pointer basics:**
- **0-reset_to_98.c**: Update pointer value to 98
- **1-swap.c**: Swap two integers

**String length:**
- **2-strlen.c**: Return string length

**String printing:**
- **3-puts.c**: Print a string
- **4-print_rev.c**: Print string in reverse
- **5-rev_string.c**: Reverse a string
- **6-puts2.c**: Print every other character
- **7-puts_half.c**: Print second half of string
- **8-print_array.c**: Print array elements

**String copy:**
- **9-strcpy.c**: Copy a string

**String concatenation:**
- **0-strcat.c**: Concatenate two strings
- **1-strncat.c**: Concatenate with n bytes

**String compare and transform:**
- **2-strncpy.c**: Copy n characters
- **3-strcmp.c**: Compare two strings
- **4-rev_array.c**: Reverse array content
- **5-string_toupper.c**: Convert to uppercase
- **6-cap_string.c**: Capitalize words
- **7-leet.c**: Encode into 1337 speak

**Memory operations:**
- **0-memset.c**: Fill memory with constant byte
- **1-memcpy.c**: Copy memory area
- **2-strchr.c**: Locate character in string
- **3-strspn.c**: Get prefix substring length
- **4-strpbrk.c**: Search string for any set of bytes
- **5-strstr.c**: Find substring

**2D arrays:**
- **7-print_chessboard.c**: Print chessboard
- **8-print_diagsums.c**: Sum diagonals of matrix

## Key Concepts

Pointers (dereference, address-of), pointer arithmetic, arrays, string reimplementation (strlen/strcat/strcmp/strcpy), memory functions (memset/memcpy/strchr/strspn/strpbrk/strstr), 2D arrays, leet encoding
